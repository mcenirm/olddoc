% olddoc(1) olddoc user manual

# NAME

olddoc - old-fashioned RDoc HTML generator

# SYNOPSIS

`olddoc` prepare

`rdoc` -f oldweb

# DESCRIPTION

olddoc features oldweb, and old-fashioned RDoc HTML generator.
You can also use "olddoc prepare" to generate NEWS files from
git tags.
